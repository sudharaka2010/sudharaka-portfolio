Admin uploads are saved here so they stay inside the project folder.

Folders:

1. `public/figma/uploads/videos`
2. `public/figma/uploads/images`
3. `public/figma/uploads/thumbnails`

Metadata file:

- `data/figma-gallery.json`

Workflow:

- Upload from the admin panel.
- The files are written into this project folder.
- Run `git add .`, commit, and push to include them in the repo.
